What is Copyleft?

什么是 Copyleft？

Copyleft is a general method for making a program or other work free, and requiring all modified and extended versions of the program to be free as well.

Copyleft是一种使一个程序或其他工作自由的一般方法，并且要求所有对该程序的修改版本和衍生版本也是自由的。

The simplest way to make a program free software is to put it in the public domain, uncopyrighted.

把你的程序发布在没有版权概念的公共社区，是使其成为自由软件的最简单方法。

This allows people toshare the program and their improvements, if they are so minded. But

如果他们足够明智，可以共享出他们的程序或是成果

it also allows uncooperative people to convert the program into proprietary software. They can make changes, many or few, and distribute the result as a proprietary product. People who receive the program in that modified form do not have the freedom that the original author gave them; the middleman has stripped it away.

他们可以或多或少做一些修改，然后作为专利软件发布，当用户使用这些软件时就享受不到原本自由软件作者的免费恩惠了， 因为这些中间人从中获取了自由软件的价值

In the GNU project, our aim is to give all users the freedom to redistribute and change GNU software.

在 GUN项目里，我们的目的是给于所有的用户去重新改造和修改GUN软件的自由

If middlemen could strip off the freedom, we might have many users, but those users would not have freedom.

如果这些中间人剥夺自由软件的理念，[那么]我们或许会拥有更多的用户， 但这些用户将享受不到有自由软件

So instead of putting GNU software in the public domain, we “copyleft” it.

所以坚持把GUN 软件发布在自由社区，我们可以copyleft 这些GUN软件

Copyleft says that anyone who redistributes the software, with or without changes, must pass along the freedom to further copy and change it. Copyleft guarantees that every user has freedom.

Copyleft 意味着任何人都可以对其改造，修改，或者原封不动，但必须传承这种自由以使人们更多的复制和修改， copyleft 会保证每个人享有这个自由

Copyleft also provides an incentive for other programmers to add to free software. Important free programs such as the GNU C++ compiler exist only because of this.

Copyleft也激励其他的程序员们去发布他们的自由软件，像比较有名的GNU C++编译器就是一个很好的例子

Copyleft also helps programmers who want to contribute improvements to free software get permission to do that.

copyleft 也能帮助那些想把自己成果作为自由软件[免费]贡献出来的程序员

These programmers often work for companies or universities that would do almost anything to get more money.

这些程序员通常为公司或大学工作,而公司或大学则尽其所能为他们提供报酬

A programmer may want to contribute her changes to the community, but her employer may want to turn the changes into a proprietary software product.

一个程序员可能希望为社区做一些改进，但他们的老板并不希望这样，而是想把它们变成有专利权的软件产品

When we explain to the employer that it is illegal to distribute the improved version except as free software, the employer usually decides to release it as free software rather than throw it away.

但我们给老板解释除了自由软件，发布其他改进版本的软件是非法的时候，通常老板都坚持把它们说成是自由软件而并不是舍弃不用

To copyleft a program, we first state that it is copyrighted; then we add distribution terms, which are a legal instrument that gives everyone the rights to use, modify, and redistribute the program's code or any program derived from it but only if the distribution terms are unchanged. Thus, the code and the freedoms become legally inseparable.

Copyleft一个程序，我们首先要确定它是否是有版权的，然后制定分布规则， 这些规则允许每一个人都有权利去使用，修改和重修改程序的代码或者其派生程序的代码,但前提是这些分布规则没有改变。 于是这些代码和自由精神也就绝对合法了

Proprietary software developers use copyright to take away the users' freedom; we use copyright to guarantee their freedom. That's why we reverse the name, changing “copyright” into “copyleft.”

专利软件的开发者利用版权剥夺用户的自由，而我们用版权赋予用户自由，这就是我们颠倒单词，把Copyright改为Copyleft>的缘故了

Copyleft is a way of using of the copyright on the program. It doesn't mean abandoning the copyright; in fact, doing so would make copyleft impossible. The word “left” in “copyleft” is not a reference to the verb “to leave” — only to the direction which is the inverse of “right”.

Copyleft 是一个使用程序版权的方法，它并不意味抛弃了版权，事实上，这样做会使Copyleft不可能实现， Copyleft里的单词left不是参考动词”to leave “的意思，只是说明正好和right相反

Copyleft is a general concept, and you can't use a general concept directly; you can only use a specific implementation of the concept.

Copyleft 是一个普遍的理念，你不能直接使用这个理念,但你可以使用其特定的概念

In the GNU Project, the specific distribution terms that we use for most software are contained in the GNU General Public License (available in HTML, text, and Texinfo format).

在GUN项目中，我们使用最多的来自一些特殊特殊规则去修改软件， 这些软件都被包含在GNU General Public License (允许使用HTML，Text，Texinfo格式）, and Texinfo format)

The GNU General Public License is often called the GNU GPL for short. There is also a Frequently Asked Questions page about the GNU GPL. You can also read about why the FSF gets copyright assignments from contributors.

The GNU General Public License 一般被简称为GUN GPL，你也可以阅读why the FSF gets copyright assignments from contributors, 常见问题主页也是讲关于GUN GPL 的， 你也可以参考why the FSF gets copyright assignments from contributors主页，这些都是讲关于GUN GPL 的

An alternate form of copyleft, the GNU Lesser General Public License (LGPL) (available in HTML, text, and Texinfo format), applies to a few (but not all) GNU libraries. To learn more about properly using the LGPL, please read the article Why you shouldn't use the Lesser GPL for your next library.

还有一种交替的Copyleft the GNU Lesser General Public License (LGPL) (available in HTML, text, and Texinfo format)，参考一些（但不全是）GUN librarys， 想了解更多的有关LGPL的特性可以看看这篇文章Why you shouldn't use the Lesser GPL for your next library

The GNU Free Documentation License (FDL) (available in HTML, text and Texinfo) is a form of copyleft intended for use on a manual, textbook or other document to assure everyone the effective freedom to copy and redistribute it, with or without modifications, either commercially or noncommercially.

GNU Free Documentation License (FDL) (可以使用 HTML, text and Texinfo格式)是用作手册，教科书或其它文档并且确保每个人都能行使复制和改进的自由， 无论有没有修改或是商业化与否的一种copyleft的延续版本

The appropriate license is included in many manuals and in each GNU source code distribution.

相关合适的许可被包含在许多指南手册上和GUN 发布的源码中

All these licenses are designed so that you can easily apply them to your own works, assuming you are the copyright holder. You don't have to modify the license to do this, just include a copy of the license in the work, and add notices in the source files that refer properly to the license.

所有的许可被设计成方便你去把他们运用到你的作品中，假定你是版权持有者，你不必为此修改证书， 仅仅把一份证书拷贝到你作品里，再向证书里添加一些源码的说明和参考引用就行了>

Using the same distribution terms for many different programs makes it easy to copy code between various different programs.

通过利用相同规则对不同程序的修订, 可以使得在各种不同的程序之间复制代码变得简单

Since they all have the same distribution terms, there is no need to think about whether the terms are compatible. The Lesser GPL includes a provision that lets you alter the distribution terms to the ordinary GPL, so that you can copy code into another program covered by the GPL.

既然都是用相同编译规则, 那就没有必要去考虑这些编译规则是否合适，仅有较少的GPL包含一个让你依据普通GPL更改团队的条款， 所以你可以依据GPL把你的代码拷贝到另一个程序中

If you would like to copyleft your program with the GNU GPL or the GNU LGPL, please see the license instructions page for advice. Please note that you must use the entire text of our licenses, if you use one. Each is an integral whole, and partial copies are not permitted.

如果你想通过GUN GPL 或者 GUN LGPL copyleft你的程序可以参考证书指南，但请务必注明完整的证书， 如果你只注明一部分，或只是其中的一部分，那么部分的拷贝将是不允许的

If you would like to copyleft your manual with the GNU FDL, please see the instructions at the end of the FDL text, and the GFDL instructions page. Again, partial copies are not permitted.

如果你想通过GUN FDL 来Copyleft 你的指南手册，你参阅PDL文本的结尾部分和GFDL 指南的说明，仍然，部分的拷贝不被允许
