Apache 介绍

Apache 官方网站：http://www.apache.org/

基本使用

安装

  • Redhat 下用压缩可执行包安装 Apache：
    • # rpm -qa|grep apache
    查看是否已经安装 Apache
    # wget http://URL-OF-Apache/apache.tar.gz
    下载 Apache 软件包
    # tar xcvf apache.tar.gz
    解压软件包
    # cd apache/
    # make
    # make install
    # cd scripts
    # apache_install_db
    完成具体编译、安装
  • Fedora 下通过 yum 安装 Apache：
    • # yum install apache
  • Ubuntu 下通过新利得安装 Apache：
    • # apt-get install apache

配置

  • 启动系统时自动运行 Apache 服务器
    • 在 /etc/rc.local中添加命令行 /etc/init.d/apache start
  • 常规配置
    • ip端口wwwroot
  • 虚拟主机配置
  • 访问控制

基本操作

  • 服务器命令：
    • #/etc/init.d/apache start
    启动 Apache 服务
    #/etc/init.d/apache stop
    关闭 Apache 服务
    #/etc/init.d/apache restart
    重启 Apache 服务
    #/etc/init.d/apache status
    查看 Apache 服务状态
    #/etc/init.d/apache reload
    重新加载 Apache 服务
    # /etc/init.d/apache force-reload
    强制重新加载 Apache 服务
