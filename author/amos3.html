编程入门学习 CodeMonkey.co.il

Date 2014-08-01 10:22

非常简单的编程学习网站： http://www.codemonkey.co.il/，通过三四个语法控制猴子和乌龟移动吃光所有香蕉，总共30关。很有趣，又能了解编程的一些基本概念，特别适合没有太多编程基础的人。

step用来移动对象，默认移动猴子。 “turtle.step 10” 用来移动乌龟10步。 turn 用于转动方向，"turn left"：向左移动90度，"turn -90": 顺时针移动90度，“turnTo bridge”：移动到bridge所在方向。 3.times -> 实现循环

要完成任务，需要躲避障碍，搭乘乌龟在河中移动，其中的距离可以使用尺子在编程时度量到。

Batch-converting raw image files to JPEG in Linux (批量转换RAW格式图片为JPEG格式)

Date 2014-07-19 06:35

Reference： http://www.mohdshakir.net/2012/03/26/batch-converting-raw-image-files-to-jpeg-in-linux

RAW格式的图片一张占十几兆空间，伤不起呀。。可以使用ufraw-batch命令批量转换称jpeg，可以通过参数调节压缩比例，一般能压缩到2M左右。

1. 安装ufraw软件包
# yum install ufraw ufraw-common

2. 批量转换RAW格式(这里文件后缀为.ARW) 文件为JPEG，保存到jpeg/子目录里
# ufraw-batch --out-type=jpeg --out-path=jpeg/ --compression=100 --overwrite *.ARW

使用nginx虚拟主机配置

Date 2014-06-13 13:53

在linux虚拟机启动一个web服务器, 开发部署想都通过普通用户. 普通用户没有权限监听80端口.

这里可以使用nginx配置虚拟主机, 把 vm.amosk.info的80端口映射到普通端口8000.

  1. 安装nginx服务, 并设置开机启动

  2. 添加nginx虚拟主机配置文件 /etc/nginx/conf.d/amos.conf

    upstream vm.amosk.info { server 127.0.0.1:8000; }

    server { listen 80; server_name vm.amosk.info;

    location /media/ { alias /home/amos/public/media/; }

    location / { proxy_pass http://vm.amosk.info; } }
